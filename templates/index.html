Features for the Project

Pot hole detection

Training of the model

We have used a whooping 1000 image data set to train the model in our custom build PC (personal computer). We have split At 9:1 ratio to train and test the model with 50000 epochs to give the best result.

SSD ResNet50 v1 Model

ResNet50 is a convolutional neural network that is 50 layers deep. The pretrained network can classify images into 1000 object categories, such as keyboard, mouse, pencil, and many animals.

Performance

We have trained the model until it reaches the lowest loss, About 99.9% of accuracy and only 0.1% Loss. It is able to effectively learn and recognize complex patterns in images, thanks to its deep convolutional architecture.

Limitations

It takes a considerable amount of time to load the model into the web app, GUI or CLI applications. it is a large and complex model, which can require a significant amount of computational resources to train and use.

About Us

Our project is a system that uses sensors and cameras to automatically detect and report potholes on roads. Potholes are a common problem on roads and can cause damage to vehicles and pose a safety hazard to drivers and passengers. By using sensors and cameras, a pothole detection system can quickly and accurately identify potholes and alert road maintenance crews to their location so that they can be repaired.
